This learning project is made to dive deeper into Tailwind CSS

`style.css` is the output file that will be generated by Tailwind CLI when I run the `npm run dev`. `style.css` will contain all the necessary CSS styles that Tailwind generates based on the classes I use in my HTML file. By linking `style.css` in my HTML file, I can apply Tailwind's utility classes to style my project effectively.
`npm run build` command will minify my Tailwind CSS codes removing whitespaces.
